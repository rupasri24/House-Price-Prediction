House Price Prediction Using Machine Learning
Project Overview

This project predicts house prices using Machine Learning techniques. The dataset contains information about house features such as area, bedrooms, bathrooms, parking, furnishing status, and other amenities.

Objective

To build a machine learning model that can predict house prices based on different house characteristics.
